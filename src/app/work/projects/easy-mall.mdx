# Easy Mall

A comprehensive e-commerce platform designed to provide a seamless shopping experience for customers while offering powerful tools for vendors and administrators.

## Key Features

- **Product Catalog**: Extensive product database with search and filtering
- **Shopping Cart**: Persistent cart with real-time updates
- **Payment Integration**: Secure payment processing with multiple gateways
- **Vendor Management**: Multi-vendor marketplace functionality
- **Order Management**: Complete order processing and tracking

## Technologies Used

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Django, Django REST Framework
- Database: PostgreSQL
- Payment: Stripe, PayPal
- Search: Elasticsearch

## Project Highlights

This platform showcases expertise in building scalable e-commerce solutions with multi-vendor support and comprehensive payment processing.
