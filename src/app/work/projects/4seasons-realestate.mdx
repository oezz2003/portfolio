# 4Seasons Real Estate

A professional real estate website developed for a real estate company using **Next.js** integrated with **WordPress** as a headless CMS through **REST API**.  
The platform delivers a visually appealing experience with fluid animations, optimized SEO, and advanced property search capabilities.

## Key Features

- **Headless CMS Integration**: WordPress used as a content management system via REST API  
- **Modern Frontend**: Built with Next.js for fast performance and smooth navigation  
- **Advanced Search Filters**: Filter properties by location, price, property type, and more  
- **Clean UI & Animations**: Elegant transitions and a polished, modern layout  
- **SEO Optimized**: Structured content, fast loading, and meta optimizations for better rankings  
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices  
- **Smart CTA Placement**: Strategically designed calls-to-action for better user conversion  

## Technologies Used

- **Frontend**: Next.js, React.js  
- **CMS**: WordPress (Headless via REST API)  
- **Styling**: Tailwind CSS  
- **Hosting**: Vercel  
- **APIs**: WordPress REST API, Google Maps API  

## Project Highlights

This project demonstrates the integration between **Next.js** and **WordPress** to build a dynamic, SEO-friendly real estate website.  
It showcases modern frontend techniques, smooth UI animations, and a well-balanced layout optimized for both users and search engines.
