# Eagle Divers

A comprehensive diving services platform designed to connect divers with professional instructors and manage diving expeditions safely and efficiently.

## Key Features

- **Dive Booking**: Online booking system for diving courses and trips
- **Instructor Management**: Profile management for diving instructors
- **Safety Tracking**: Dive logs and safety compliance monitoring
- **Equipment Management**: Gear rental and maintenance tracking
- **Community Features**: Diver profiles and social interactions

## Technologies Used

- Frontend: Next.js, TypeScript
- Backend: Express.js, Node.js
- Database: MongoDB
- Maps: Google Maps for dive locations
- Calendar: FullCalendar integration

## Project Highlights

This platform showcases expertise in building specialized service platforms with industry-specific features and safety compliance systems.
